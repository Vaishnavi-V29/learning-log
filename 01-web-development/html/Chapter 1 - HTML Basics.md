# 📘 Chapter 1 — HTML Basics

## 📌 DOCTYPE
Declares that the document is HTML5 and tells the browser to render it in modern standard mode.
---

## 🏷️ Tag
A keyword inside angle brackets used to define structure in HTML.
Example: `<p>`, `<html>`, `<body>`
---

## 🧱 Element
A complete structure made of:
* opening tag
* content
* closing tag
Example: `<p>Hello</p>`
---

## Attributes of a tag
Provide additional information to a tag.
Example: `<a href="https://google.com">`
---

## 🧠 `<head>`
Contains **non-visible information** about the webpage:
* metadata
* title
* links
* scripts
* settings
---

## 👀 `<body>`
Contains **visible content** shown to the user:
* text
* images
* buttons
* layout
